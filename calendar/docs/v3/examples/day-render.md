# Day Render

A hook for modifying a day DOM.

## Usage

```js
new HelloWeek({
  selector: '.calendar',
  dayRender: data => {
    data.node = el('div', data.attributes, el('span', {}, data.day.toString()));
    return data;
  }
});
```
